# Unstop Certificate Email Automation Bot

## 🚀 Overview
This project automates certificate generation and email distribution for events hosted on Unstop.

## ✨ Features
- Works directly with Unstop CSV exports
- Filters only paid participants
- Automatically generates personalized certificates
- Sends professional email with attachment
- Multi-event folder structure
- Secure credential handling using .env

## 🛠 Tech Stack
- Python
- Pandas
- Pillow
- Yagmail

python3 main.py event_name
